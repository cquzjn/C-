类、字段、属性、方法：

类中的成员分为字段、属性和方法，方法即函数，字段为各种变量和常量，属性较为特殊，是用来限制字段的可访问级别的，由 Set 或 Get 函数组成。

同一命名空间下的类默认可以互相访问，不同类下的字段，默认不可以互相访问（若要互相访问，需加Public关键字）。 类通常不嵌套使用，因为类是共享的，除非该类仅限定在特定的类中使用，作为private的类时，可以嵌套使用。

构造函数：

构造函数是类在实例化时自动调用的方法，通常在其中写入初始化函数（以及类中需要实例化的成员）。

类的继承：

子类可以继承父类中的所有成员，并且可以定义新的成员。当对子类进行实例化时，首先调用父类的构造函数，随后调用子类的构造函数。
